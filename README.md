# AI Client Assistant

Мой первый проект на Python.

## Что делает программа

- принимает имя пользователя
- принимает ID пользователя
- определяет VIP или обычный клиент
- генерирует случайное сообщение
- сохраняет всех пользователей
- выводит статистику

## Как работает

- чётный ID → VIP
- нечётный ID → обычный пользователь

## Как запустить

1. открыть файл main.py
2. запустить Python
3. вводить данные
