# AI Client Assistant

Simple Python application for user segmentation and personalized message generation  

Мини-программа для сегментации пользователей и генерации персональных сообщений

---

## 🚀 Features

- User input  
- VIP detection  
- Random message generation  
- Statistics calculation  

---

## 🚀 Что делает программа

- принимает имя пользователя  
- принимает ID пользователя  
- определяет VIP или обычный клиент  
- генерирует случайное сообщение  
- сохраняет всех пользователей  
- выводит статистику  

---

## ⚙️ How it works

- even ID → VIP user  
- odd ID → regular user  

---

## ⚙️ Как работает

- чётный ID → VIP  
- нечётный ID → обычный пользователь  

---

## ▶️ How to run

1. Open `main.py`  
2. Run Python  
3. Enter user data  

---

## ▶️ Как запустить

1. открыть файл `main.py`  
2. запустить Python  
3. вводить данные
